# Frontend README

## 🎨 Interface do Sistema Pastoral Escolar

Frontend em React com TailwindCSS.

## 📋 Requisitos

- Node.js 18+
- npm ou yarn

## 🛠️ Instalação

\`\`\`bash
# Instalar dependências
npm install

# Iniciar desenvolvimento
npm run dev
\`\`\`

## 🎨 Características

- **React 18** com hooks
- **TailwindCSS** para estilização
- **React Router** para navegação
- **Lucide React** para ícones
- **Design responsivo**
- **Menu lateral fixo**
